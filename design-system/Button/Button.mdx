### Usage

Check some examples of how to use the Button component

Primary Button

```.jsx
<Button handleClick={() => console.log('clicked')}>This is a Button</Button>
```

Secondary Button

```.jsx
<Button
  handleClick={() => console.log('clicked')}
  mode="secondary"
>
  This is a Button
</Button>
```
